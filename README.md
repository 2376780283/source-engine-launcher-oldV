# srceng-android-launcher-

## 这是一个起源引擎的不完全开源的安卓端的启动器
### 开发中... beta的屎

## 编译指南
本项目使用Android IDE 和android studio开发开发 建议也使用此工具进行维护

#注意事项
本项目拥有完整的 构建 而不是像nillerusr 只是做了一个简单的继承活动
这个项目是完全使用 androidx 并且包名不能随便改
如果需要更改按钮样式 使用android.widget.button
